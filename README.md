# Mel en off ✈️

Landing page — carnet d'adresses voyage (hôtels & restaurants) avec liens d'affiliation.

- Page statique autonome : **`index.html`** (photos intégrées en base64, aucune dépendance).
- Déploiement : Vercel (import du repo GitHub).
- Contact / collabs : **contact@melnourdi.fr**

## Éditer
- Remplacer une photo : `<img class="photo" src="..." >`
- Lien d'affiliation : remplacer `href="#"`
- Nouvelle adresse : dupliquer un bloc `<article class="card">`
